## Requirements

- Node 6.9.1
- NPM
- React 15.4.1
- MySQL 2.12.0
- Express 4.14.0
- Sequelize 3.27.0

## Development

### Installing Dependencies

From within the root directory:

```sh
$ npm install
$ mysql.server start
 >> $ mysql -u root -p    #(hit enter when prompted for password)
 >> $ create database crashcourse
$ grunt
```
